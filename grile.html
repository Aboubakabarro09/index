historique et introduction de swift

Swift est un langage de programmation développé par Apple. Il est développé en open source.(logiciel libre) Swift dispose d'une capacité à fonctionner efficacement avec le langage C ainsi qu'avec l'Objective-C qui est un langage de programmation objet fixe sur les plateformes d'Apple. Le développement de Swift a été commencé en 2010 par Chris Lattner et prend ses sources dans de nombreux langages de programmation tels qu'Objective-C, Haskell, Ruby, Python, Caml ou C#. La première version publique (1.0) a été publiée le 2 juin 2014 en Californie aux Etats-Unis.

En juin 2015, lors de l’Apple Worldwide Developers Conference (WWDC), apple annonce le passage en open-source du langage swift avec la distribution du code du compilateur et des bibliothèques standard. Il fut la même année en décembre.


Fonctionnalités

Swift inclut des fonctionnalités qui facilitent la lecture et l’écriture du code, tout en donnant au développeur le contrôle nécessaire dans un véritable langage de programmation système.

Swift prend en charge les types inférés pour rendre le code plus propre et moins sujet aux erreurs, et les modules éliminent les en-têtes et fournissent des espaces de noms. La mémoire est gérée automatiquement et vous n’avez même pas besoin de taper des points-virgules.

Les fonctionnalités de Swift sont conçues pour fonctionner ensemble afin de créer un langage puissant et amusant à utiliser.


Securité

Swift a été conçu dès le départ pour être plus sûr que les langages C et éliminer des classes entières de code dangereux. Les variables sont toujours initialisées avant utilisation, les tableaux et les entiers sont vérifiés pour le dépassement de capacité et la mémoire est gérée automatiquement. La syntaxe est réglée pour faciliter la définition de votre intention — par exemple, de simples mots-clés à trois caractères définissent une variable () ou une constante ().varlet


Plateformes supportées

Swift est officiellement supporté sur les plateformes d'Apple, sur Linux (Ubuntu, CentOS 8, Amazon Linux 2)21 et Windows (depuis la version 5.3).

Le support est assuré de manière non officielle pour Android, Linux sur l'architecture ARM, Windows, WebAssembly et d'autres systèmes. L'équipe responsable du développement de Swift a annoncé vouloir supporter officiellement de nouvelles plateformes.

En ce jour, il est considéré comme le langage de programmation le plus utilisé.

installation

sur pc
  1. Aller sur votre navigateur et taper "Apple developer";
  2. Arriver sur la page d'accueil de la plateformme, Cliquer sur develop qui se trouve sur la barre verticale de la page d'accueil;
  3. Ensuite, cliquer sur Xcode juste sous la barre verticale;
  4. sur la même ligne, appuyer sur "Download ou télécharger"
  5. Il vous sera demander de vous identifier à l'aide de votre compte apple. si vous n'en avez, pas vous pouvez ouvrir votre compte appelé identifiant apple.
  6. Et vous pouvez commencer à creer tout nouveau projet
